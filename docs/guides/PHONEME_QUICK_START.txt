🎯 Goal: Get Turkish phoneme generation working in 5 minutes

───────────────────────────────────────────────────────────────────
📋 Prerequisites
───────────────────────────────────────────────────────────────────

✓ Backend dependencies already installed (phonemizer in requirements.txt)
✓ eSpeak-NG must be installed on your system

───────────────────────────────────────────────────────────────────
🔧 Step 1: Install eSpeak-NG (One-time)
───────────────────────────────────────────────────────────────────

Windows:
--------
choco install espeak-ng

macOS:
------
brew install espeak

Linux:
------
sudo apt-get install espeak-ng

Verify:
-------
espeak-ng --version
# Should show version 1.51 or higher

───────────────────────────────────────────────────────────────────
🚀 Step 2: Start Backend
───────────────────────────────────────────────────────────────────

cd backend
python main.py

Expected output:
✓ Model loaded successfully (or warning if not trained yet)
✓ eSpeak-NG backend initialized for Turkish (tr)
INFO: Uvicorn running on http://0.0.0.0:8000

───────────────────────────────────────────────────────────────────
🎨 Step 3: Start Frontend
───────────────────────────────────────────────────────────────────

# Open new terminal
cd frontend
npm run dev

Open: http://localhost:3000

───────────────────────────────────────────────────────────────────
🧪 Step 4: Test via Browser
───────────────────────────────────────────────────────────────────

1. Click "Fonem Önizleyici" button
2. Enter word: pencere
3. Click "Fonem Oluştur"
4. See phonemes: p e n d͡ʒ e ɾ e

✅ SUCCESS! Feature is working.

───────────────────────────────────────────────────────────────────
🧪 Step 5: Test via curl (Optional)
───────────────────────────────────────────────────────────────────

curl -X POST http://localhost:8000/phoneme/generate \
  -H "Content-Type: application/json" \
  -d '{"word": "merhaba"}'

Expected:
{
  "word": "merhaba",
  "phonemes": "m e ɾ h a b a",
  "phoneme_count": 7,
  "language": "tr",
  "backend": "espeak-ng"
}

───────────────────────────────────────────────────────────────────
📚 API Endpoints
───────────────────────────────────────────────────────────────────

POST /phoneme/generate    - Generate phonemes for single word
POST /phoneme/analyze     - Detailed analysis with syllables
POST /phoneme/batch       - Process multiple words
GET  /phoneme/health      - Check service status

Full docs: http://localhost:8000/docs

───────────────────────────────────────────────────────────────────
🔍 Troubleshooting
───────────────────────────────────────────────────────────────────

Problem: "eSpeak-NG backend not available"
Solution: Install eSpeak-NG (see Step 1) and restart backend

Problem: "CORS error in browser console"
Solution: Ensure backend is running on port 8000

Problem: Phonemes look incorrect
Solution: Verify Turkish support: espeak-ng --voices=tr

Problem: Frontend doesn't show phoneme button
Solution: Clear browser cache and refresh

───────────────────────────────────────────────────────────────────
🎓 Using in ML Training
───────────────────────────────────────────────────────────────────

import requests

# Get reference phonemes
response = requests.post(
    'http://localhost:8000/phoneme/generate',
    json={'word': 'kitap'}
)
phonemes = response.json()['phonemes']
print(phonemes)  # k i t a p

# Batch processing
words = ['masa', 'sandalye', 'pencere']
response = requests.post(
    'http://localhost:8000/phoneme/batch',
    json=words
)
batch_phonemes = response.json()
